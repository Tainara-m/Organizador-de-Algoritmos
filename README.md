# Organizador-de-Algoritmos
Organizador de Algoritmos: app web para aprender lógica organizando cards arrastáveis em sequência. Suporta categorias Pseudocódigo (com preview automático) e Sequencial, modo claro/escuro, atalhos de teclado, acessibilidade, e biblioteca de tarefas com importar/exportar e adição por formulário (com confirmação). Tudo em HTML/CSS/JS + localStorage.
